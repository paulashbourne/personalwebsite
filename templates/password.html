Password Manager

A simple utility to securely use the same password everywhere.

1. Enter the primary domain of the website. For example, for Facebook, enter "facebook.com"
2. Enter your password that you'd like to use
3. Click Generate Hash
4. Copy the resulting hash into the password field for the website

How does this work? The domain of the website is appended to your desired password. The resulting string is then hashed using SHA-256. The result is that you have a unique password for the website you are using. Everything is done client-side, so your credentials never leave your computer.
