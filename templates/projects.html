Projects

Here is a short list of some of the cool projects that I have worked on.

Health Care Patient Database

Developed an application and database to store health care records for thousands of palliative care patients in the Greater Toronto Area. Independently built the system from the ground-up to replace an existing Microsoft Access application that suffered from poor performance and excessive complexity. Extracted all data from original database and transferred to new database while maintaining data integrity. Currently holds over 100,000 records. Set up VPN Server for secure access. Uses MySQL and PHP backend.
[MySQL, PHP, VB.NET, HTML+CSS, JavaScript+jQuery]

Developed for ESCTT Inc. during employment as Software Engineer.

RiskMP Risk Management Software

A fully-featured risk management web application that provides project managers with tools to analyze the impact of risks that may impact projects and create mitigation plans. Engineered the database, all application logic and the user interface. Data model allows users to grant specific permissions to other users to enable collaboration and project sharing. Includes automatic recurring billing system. Developed an extension for Microsoft Project that allows users to synchronize their projects. Beta testing conducted by project management professionals has yielded excellent feedback.
[MySQL, PHP, VB.NET, HTML+CSS, JavaScript+jQuery]

Developed for ESCTT Inc. during employment as Software Engineer. - Visit Site

Financial Management Application

Built a web application that provides financial advisors and investment professionals with tools to conduct in-depth analysis of client portfolios and track performance in compliance with CRM2 standards. Developed data model that allows for analysis of multiple portfolios across various clients. Early feedback from financial professionals has been extremely positive.
[MySQL, PHP, Python, HTML+CSS, JavaScript+jQuery]

Developed for ESCTT Inc. during employment as Software Engineer.

Remote Desktop Application

Coded in Java, this program consists of both client and server applications. When deployed, the software allows a user to connect to a remote computer and view the display while controlling the mouse and keyboard inputs. Only the core functionality was developed; the user interface is minimal and there are no additional features. Uses threading to detect pixel changes in different areas of the display.
[Java, Threading]

Live Television Display Application

Developed as my final assignment for Grade 12 Computer Science, this application accesses RSS feeds to display updated weather and news reports. Also displays a slideshow of /static/img in a user-selected folder. Fully-customizable. Currently in use on all television displays in Aurora High School.
[Java]

Sudoku Solver Application

Developed an application that can solve any Sudoku puzzle in the blink of an eye. Uses a modified version of the backtracking algorithm for optimal performance.
